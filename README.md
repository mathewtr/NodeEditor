Follow these steps to run the editor locally on your machine.

Prerequisites

Node.js (v16 or higher)

npm or yarn

Installation

Clone the repository:

git clone [url]
cd folderName


Install dependencies:

npm install


Start the development server:

npm run dev


Open your browser to the local URL (usually http://localhost:5173 or http://localhost:3000).
